### 🔬 [Псофометрический фильтр BS.468-4 / Psophometric Filter Design](https://github.com/AlexKlubyshev/psophometric_filter_design)
**Проектирование цифрового фильтра методом машинного обучения**

![Frequency Response](https://raw.githubusercontent.com/AlexKlubyshev/psophometric_filter_design/main/images/af_468.png)

**Задача:** Спроектировать КИХ-фильтр, соответствующий стандарту BS.468-4 (взвешивание шумов по чувствительности человеческого уха)<br>
**Domain:** Цифровая обработка сигналов (ЦОС/DSP), Радиочастотная инженерия, Аудиотехнологии, Метрология<br>
**Уникальность подхода:** Впервые коэффициенты фильтра получены через **Ridge-регрессию**, а не классическими методами (частотная выборка, Ремез)<br>
**Стек:** Python, Scikit-learn (RidgeCV), NumPy, SciPy, Matplotlib

**Ключевые результаты:**
- ✅ Фильтр 127 порядка с R² = 0.95 на тестовых данных
- ✅ Полное соответствие допускам стандарта в диапазоне 6.3–10 кГц
- ✅ Выявлено ограничение последовательной коррекции (фазовые искажения до -500°)

**Что исследовано:**
- Сравнение фильтров 63 и 127 порядка (улучшение на 50% в допусках)
- Анализ применимости ML для задач цифровой обработки сигналов
- Влияние порядка фильтра на точность воспроизведения АЧХ/ФЧХ

**Цель проекта:** Исследовать применимость ML методов для проектирования цифровых фильтров взамен классических подходов (частотная выборка, метод Ремеза)<br>
**Архитектурная ценность:** ML-based filter design pipeline с масштабируемостью под произвольные стандарты<br>
**Связь с опытом:** Применение радиоинженерного образования (СПбГУТ) + современные ML методы в задачах ЦОС
